Lịch Sử Python

Hành trình từ sở thích cá nhân đến ngôn ngữ số 1 thế giới

1989

Guido van Rossum bắt đầu phát triển

Tại Trung tâm Toán học và Tin học Hà Lan (CWI), Guido bắt đầu viết Python như một dự án cá nhân trong kỳ nghỉ Giáng Sinh.

1991

Python 0.9.0 ra mắt

Phiên bản đầu tiên được công bố, đã có các tính năng cốt lõi như class, exception handling và functions.

2000

Python 2.0

Hỗ trợ Unicode, garbage collection và list comprehensions. Python bắt đầu được cộng đồng quốc tế đón nhận rộng rãi.

2008

Python 3.0 — Bước đột phá

Phiên bản tái thiết kế hoàn toàn với cú pháp rõ ràng hơn, không tương thích ngược với Python 2.

2020+

Python thống trị kỷ nguyên AI

Với sự bùng nổ của AI và Data Science, Python trở thành ngôn ngữ không thể thiếu, đứng đầu mọi bảng xếp hạng lập trình.
